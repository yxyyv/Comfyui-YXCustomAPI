# 简介
YXCUSTOMAPI用于支持第三方API接入生图模型到Comfyui中。

## 当前支持的模型
- Nano Banana 2
- Nano Banana pro
- GPT Image 2 (1k版本)

---
# 注意事项
- 需要支持comfy_api.latest的comfyui版本[(v3)](https://docs.comfy.org/custom-nodes/v3_migration)
- 当前测试comfyui最低版本：v0.19.3
- 节点超时设置：“-1”代表不设超时，“0”代表自动
- 使用前先在 provider.json 中配置好api供应商，且确保激活的供应商“enabled”状态为1。
